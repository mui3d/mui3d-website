# Mui3D Portfolio

A modern, dark-themed 3D portfolio website for Mui3D.

## Pages

- **Home** — Interactive Three.js hero with animated 3D geometry
- **About** — Bio, skills, and stats
- **Portfolio** — Filterable project gallery
- **YouTube** — Embedded video showcase
- **Contact** — Contact form and social links

## Getting Started

Open `index.html` in a browser, or serve locally:

```bash
# Python
python -m http.server 8080

# Node (if installed)
npx serve .
```

Then visit `http://localhost:8080`.

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Three.js](https://threejs.org/) (CDN) for 3D hero scene
- Google Fonts (Inter + Space Grotesk)
- Mobile-responsive with hamburger navigation

## Customization

- Replace placeholder portfolio thumbnails with real project images
- Update YouTube embed URLs with your actual video IDs
- Edit contact email and social links in `contact.html`
- Adjust colors in `css/styles.css` (`:root` variables)
